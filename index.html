Font Properties

Although some of the properties sound similar to text properties, think of font properties as those that change the style of the font on the page its family, size, weight, and related characteristics.

Font Properties Covered in Your zyBook

font-family
Define the fonts you want to use. List more than one, and keep them visually similar so if one fails, the next is a good fallback.
font-size
Controls the size of text on your page. Browsers apply defaults, so you’ll notice right away which elements are larger (headings) or smaller (paragraphs).
font-weight
Makes fonts bolder or lighter. Headings are usually bold; body text is normally regular/medium. Avoid text that’s so light or so small it’s unreadable.
font-style
Used for italic, oblique, or normal styling.
font-variant
Specifies variants of the font such as small-caps or all-uppercase presentation.
font (shorthand)
Shorthand that sets (in this order) style, variant, weight, size (and optional line-height), and family in one declaration.

Using Web Fonts

There are many ways to use fonts on the web: you can host your own or link to a service like Google Fonts. We’ll use Seymour One for headings and Lato for body text. In Chrome, go to the font “Seymour One” on Google Fonts.

  1. Choose the fonts and styles (Seymour One 400; Lato 400 and 700) and add them to the selection.
  2. Copy the generated <link> element from the sidebar.
  3. Paste that link into the <head> of your HTML above your CSS link.
  4. In your CSS, set your font stacks with font-family where needed.

Font Sizing

There are many units you can use for sizing. Keep these accessibility notes in mind:

  1. Prefer relative units (rem/em). Fixed pixels can harm zoom and device adaptability.
  2. Make sure body text is at least 1rem (about 16px) for readability.

Text Properties

Text properties change the way the layout of text appears—alignment, decoration, casing, indentation, and spacing for comfortable reading.

Text Properties Covered in Your zyBook

text-align
Controls text alignment: left, right, center, or justify.
text-decoration
Shorthand controlling decoration line, style, and color (e.g., underlines on links).
text-transform
Converts text to uppercase, lowercase, or capitalized forms.
text-indent
Indents the first line of a paragraph; helpful for scanning new paragraphs when you’re not adding extra spacing.

Line Height and Width for Text – Best Practices

These properties are important for readability and accessibility (and therefore SEO). Follow the guidelines below:

  • Use a line-height greater than 1; 1.5 is a common setting.
  • Limit paragraph width to about 45–90 characters (aim ~55–65).
  • Use the ch unit to constrain line length. It naturally relates to the number of characters, improving readability!
